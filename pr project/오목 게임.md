# 오목 게임

## 프로젝트 소개

2명이 오목을 할 수 있는 콘솔 기반 오목 게임입니다.

사용자는 좌표를 입력하여 돌을 놓을 수 있으며, 프로그램은 승패 판정, 유효한 수 검사, 무승부 판정 등을 수행합니다.

## 주제 선정 동기

수업시간에 배운 재귀함수와 조건문, 2차원 배열을 더 깊이 있게 이해하고 직접 구현해보고자 이 프로그램을 개발하였습니다.

## 주요 기능

### 1. 승패 판정

가로, 세로, 대각선 방향으로 돌이 정확히 5개 연속으로 놓였는지 검사하여 승패를 판정합니다.

### 2. 유효한 수 검사

플레이어가 입력한 좌표가 다음 조건을 만족하는지 확인합니다.

* 오목판 범위를 벗어나지 않았는가
* 이미 돌이 놓인 위치가 아닌가
* 금수(삼삼)에 해당하지 않는가

조건을 만족하지 않을 경우 해당 위치에는 돌을 둘 수 없습니다.

### 3. 오목판 출력

19×19 크기의 오목판을 콘솔에 출력합니다.

출력 기호는 다음과 같습니다.

| 기호 | 의미      |
| -- | ------- |
| .  | 빈 칸     |
| ●  | Player1 |
| ○  | Player2 |

### 4. 무승부 판정

승패가 결정되지 않은 상태에서 오목판이 모두 채워지면 무승부를 선언합니다.

## 게임 규칙

* Player1은 흑돌(●)을 사용합니다.
* Player2는 백돌(○)을 사용합니다.
* 플레이어는 번갈아 가며 돌을 놓습니다.
* 정확히 5목이 완성되면 승리합니다.
* Player1은 삼삼 위치에 돌을 둘 수 없습니다.

## 입력 방법

플레이어는 다음 형식으로 좌표를 입력합니다.

```text
y x
```

예시

```text
3 5
```

위 입력은 y=3, x=5 위치에 돌을 놓는다는 의미입니다.

## 예외 처리

다음과 같은 경우 돌을 둘 수 없습니다.

* 범위를 벗어난 위치
* 이미 돌이 놓인 위치
* 삼삼 금수 위치

각 경우에 맞는 안내 메시지가 출력됩니다.

## 사용 예시

```text
player1 입력(y x): 3 3
player2 입력(y x): 4 4
```

## 사용된 주요 개념

* 재귀 함수
* 조건문
* 반복문
* 2차원 배열
* 함수 분할
* 오목 승리 조건 검사 알고리즘
